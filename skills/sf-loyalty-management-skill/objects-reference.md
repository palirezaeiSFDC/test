Canonical list of loyalty-related objects for assessment, analysis, and verification. Standard objects link to Salesforce API documentation. Use with [org-context.md](org-context.md) and the **sf-org-agent** to verify object existence in the connected org.

## Usage for Assessment and Verification

1. **Object lookup:** Use the API Name column when writing SOQL, Apex, or metadata.
2. **Documentation:** For standard objects, follow the Documentation link for field definitions and relationships.
3. **Verification:** Use the **sf-org-agent** with `sf sobject describe -s <API Name> -o <alias>` to verify object existence in the connected org.
4. **Custom objects:** Verify with org-context.md and sf-org-agent; no external documentation exists.
