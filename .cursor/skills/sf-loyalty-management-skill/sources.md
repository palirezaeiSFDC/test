## Extracted Reference Content

All content from the official Salesforce Loyalty API PDF and the Loyalty Management Objects CSV has been extracted into the `.md` files within this skill directory:

- **[api-reference.md](api-reference.md)** — API class signatures, REST endpoints, invocable actions, picklist fields, metadata types
- **[objects-reference.md](objects-reference.md)** — Complete object list (61 standard + 13 custom) with Salesforce documentation URLs

Use these `.md` files instead of the original PDF or CSV. For complete field definitions not covered in `api-reference.md`, use the online [Apex Reference](https://developer.salesforce.com/docs/atlas.en-us.loyalty.meta/loyalty/loyalty_apex_reference.htm) linked in the **Loyalty API PDF** row above.
